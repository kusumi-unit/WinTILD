# WinTILD

WinTILD is open-source software for detecting a Transient Increase of the effective Local Diffusion (TILD) in the 2D single-molecule trajectory data. See https://doi.org/10.1101/2021.10.26.465864 for details of the method. WinTILD is distributed under the terms of the GNU GPLv3 license.

## Development Environment

- Microsoft Visual Studio 2017 (Version 15.9.22)
  - Visual C++ 2017 

## How to Use

- See /doc/WinTILDQuickManual.pdf. Sample data in the manual are available at /sample directory.
